---
title: "Get started with MOSS-TTS-Nano"
description: "Learn how to install MOSS-TTS-Nano, set up a Python environment, and generate your first voice-cloned speech output in under five minutes."
---

By the end of this guide you will have a working Python environment, the MOSS-TTS-Nano package installed, and a generated audio file produced by voice clone inference. The entire setup takes about five minutes on a machine with a working internet connection.

<Steps>
  <Step title="Create a Python environment">
    Create a clean conda environment with Python 3.12 and activate it:

    ```bash
    conda create -n moss-tts-nano python=3.12 -y
    conda activate moss-tts-nano
    ```
  </Step>

  <Step title="Clone the repository and install dependencies">
    Clone the repository, then install the Python dependencies and the package itself in editable mode:

    ```bash
    git clone https://github.com/OpenMOSS/MOSS-TTS-Nano.git
    cd MOSS-TTS-Nano
    pip install -r requirements.txt
    pip install -e .
    ```

    Installing with `pip install -e .` registers the `moss-tts-nano` command in your environment so you can call it from any directory.

    <Note>
      If `WeTextProcessing` fails to install from `requirements.txt`, install `pynini` through conda first and then install the package manually:

      ```bash
      conda install -c conda-forge pynini=2.1.6.post1 -y
      pip install git+https://github.com/WhizZest/WeTextProcessing.git
      ```
    </Note>
  </Step>

  <Step title="Run voice clone inference">
    Generate speech from text using a bundled reference audio file:

    ```bash
    moss-tts-nano generate \
      --prompt-speech assets/audio/zh_1.wav \
      --text "欢迎关注模思智能、上海创智学院与复旦大学自然语言处理实验室。"
    ```

    `--prompt-speech` points to the reference audio that the model uses to clone the speaker's voice. `--text` is the text to synthesize.
  </Step>

  <Step title="Find the output file">
    The generated audio is written to:

    ```
    generated_audio/moss_tts_nano_output.wav
    ```

    Open the file in any audio player. It is 48 kHz stereo WAV audio.
  </Step>
</Steps>

<Tip>
  Model weights are downloaded automatically from Hugging Face the first time you run inference. The TTS model is fetched from `OpenMOSS-Team/MOSS-TTS-Nano` and the audio tokenizer from `OpenMOSS-Team/MOSS-Audio-Tokenizer-Nano`. No manual download step is required.
</Tip>
