MOSS-TTS-Nano is an open-source multilingual speech generation model from [MOSI.AI](https://mosi.cn) and the [OpenMOSS team](https://github.com/OpenMOSS). With only 0.1 billion parameters, it generates high-quality 48 kHz stereo audio in real time — even on a 4-core CPU — using a pure autoregressive Audio Tokenizer + LLM pipeline.

## Why MOSS-TTS-Nano?

MOSS-TTS-Nano is designed for developers who need a lightweight, deployable TTS model without GPU requirements or complex infrastructure. It focuses on three things: small footprint, low latency, and simple setup.

## Get started

<Steps>
  <Step title="Install the package">
    Clone the repository and install with pip in editable mode to get the `moss-tts-nano` CLI.

    ```bash
    git clone https://github.com/OpenMOSS/MOSS-TTS-Nano.git
    cd MOSS-TTS-Nano
    pip install -r requirements.txt
    pip install -e .
    ```
  </Step>
  <Step title="Generate speech from the CLI">
    Run voice cloning with a built-in reference audio file. The model downloads automatically from Hugging Face on first run.

    ```bash
    moss-tts-nano generate \
      --prompt-speech assets/audio/zh_1.wav \
      --text "欢迎关注模思智能、上海创智学院与复旦大学自然语言处理实验室。"
    ```
  </Step>
  <Step title="Launch the web demo (optional)">
    Start the local FastAPI web server to interact with the model via a browser UI.

    ```bash
    moss-tts-nano serve
    ```

    Then open `http://127.0.0.1:18083` in your browser.
  </Step>
</Steps>
