MOSS-TTS-Nano supports **20 languages** out of the box. You select the language by passing the appropriate language code with your input text. Voice quality and naturalness may vary by language depending on training data distribution.

## Text normalization

<Note>
  Automatic text normalization via [WeTextProcessing](https://github.com/WhizZest/WeTextProcessing) is currently supported for **Chinese (`zh`) and English (`en`) only**. For all other languages, text is passed through as-is. If you need normalization for another language, pre-process your text before passing it to the model.
</Note>

The normalization pipeline handles number expansion, abbreviation handling, and special character conversion before the text reaches the LLM tokenizer.

## Built-in voice presets

<Tip>
  The default voice is `Junhao`. To use a different built-in preset from Python, pass the `voice` parameter to `synthesize` or `synthesize_stream`. From the CLI, built-in presets are selected by passing their audio file path to `--prompt-speech` (e.g. `--prompt-speech assets/audio/en_2.wav` for Ava). To clone any other voice, provide your own reference audio file via `--prompt-speech`.
</Tip>
